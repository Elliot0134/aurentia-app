# Icônes de Livrables

Ce dossier contient les icônes utilisées pour les différents types de livrables dans l'application.

## Icônes requises

Les fichiers PNG suivants doivent être placés dans ce dossier :

1. `market-icon.png` - Analyse de marché & Mini SWOT
2. `retranscription-icon.png` - Retranscription concept
3. `story-icon.png` - Ma Success Story
4. `ressources-icon.png` - Analyse des ressources
5. `persona-icon.png` - Persona Express
6. `vision-icon.png` - Vision, Mission, Valeurs
7. `juridique-icon.png` - Cadre Juridique
8. `proposition-valeur-icon.png` - Proposition de valeur
9. `business-model-icon.png` - Business Model
10. `pitch-icon.png` - Pitch
11. `concurrence-icon.png` - Analyse de la concurrence

## Format recommandé

- Format : PNG avec transparence
- Taille : 48x48px (ou proportionnel, l'affichage est en `w-12 h-12`)
- Style : Cohérent avec le design system Aurentia

## Utilisation

Les icônes sont référencées dans les composants livrables avec le chemin `/icones-livrables/{nom-icon}.png`.

## Note

Sans ces icônes, les cartes de livrables ne s'afficheront pas correctement en production sur Netlify.
